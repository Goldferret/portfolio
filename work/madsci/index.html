MADSci MCP Server

Active — 2025–Present
Multi-agent LLM Robotics MCP Protocol

MADSci MCP is a three-agent LLM orchestration system built on top of the MADSci laboratory automation framework. The Main Agent receives a natural-language experiment brief and coordinates two specialist subagents: a Detector Agent that runs computer vision pipelines to locate and classify objects, and an Executor Agent that translates detections into physical robot actions via the Model Context Protocol.

The system was demonstrated live at a full-day MEDAL grant workshop (Argonne National Labs deliverable) and is currently being used to run a 150-trial block-sorting experiment for an IROS 2026 Workshop paper comparing tri-agent, single-agent, and scripted baselines.
